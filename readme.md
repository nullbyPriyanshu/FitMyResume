FitMyResume is a minimal AI-powered resume tailoring tool. Upload your PDF résumé, paste a job description, and instantly get optimized ATS-friendly bullet points and a concise professional summary.

## Features
- Upload a PDF résumé  
- Paste any job description  
- Generates tailored bullet points  
- Produces a short AI-optimized summary  
- Powered by **Ollama** running local AI models  
- Includes a lightweight **PDF parser** for text extraction.
